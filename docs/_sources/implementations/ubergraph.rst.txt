Ubergraph Endpoint
==================

About
-----

`Ubergraph <https://github.com/INCATools/ubergraph>`_ is a SPARQL
endpoint serving multiple OBO ontologies, pre-processed with

- relation-graph
- information content scores
- biolink categories
