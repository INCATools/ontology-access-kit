Lexical Index

A datamodel for representing a lexical index of an ontology. A lexical index is keyed by optionally normalized terms.

URI: https://w3id.org/oak/lexical-index

Name: lexical-index

Class

Description

Activity

Generic grouping for any lexical operation

An atomic lexical transformation applied on a term (string) yielding a transformed string
