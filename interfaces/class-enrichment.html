Class Enrichment Calculation Interface

class oaklib.interfaces.class_enrichment_calculation_interface.ClassEnrichmentCalculationInterface(resource: typing.Optional[oaklib.resource.OntologyResource] = None, strict: bool = False, multilingual: typing.Optional[bool] = None, preferred_language: str = <factory>, autosave: bool = <factory>, exclude_owl_top_and_bottom: bool = <factory>, ontology_metamodel_mapper: typing.Optional[oaklib.mappers.ontology_metadata_mapper.OntologyMetadataMapper] = None, _converter: typing.Optional[curies.api.Converter] = None, auto_relax_axioms: typing.Optional[bool] = None, cache_lookups: bool = False)[source]

An interface that provides services to test for over representation of class membership in a set of entities.

This interface is intended to be used to test for over representation of a class in a set of entities,

for example: to test for over representation of a disease in a set of genes.

enriched_classes(subjects: Iterable[str], predicates: Optional[Iterable[str]] = None, object_closure_predicates: Optional[List[str]] = None, background: Optional[Iterable[str]] = None, hypotheses: Optional[Iterable[str]] = None, cutoff=0.05, autolabel=False, sort_by: Optional[str] = None, direction='greater') → Iterator[oaklib.datamodels.class_enrichment.ClassEnrichmentResult][source]

Test for overrepresentation of classes in a set of entities.

Parameters

  • subjects – The set of entities to test for overrepresentation of classes

  • background – The set of entities to use as a background for the test

  • hypotheses – The set of classes to test for overrepresentation

  • cutoff – The threshold to use for significance

  • labels – Whether to include labels for the classes

  • direction – The direction of the test. One of ‘greater’, ‘less’, ‘two-sided’

Returns

An iterator over ClassEnrichmentResult objects
